Como Funciona o Radix Sort?

O Radix Sort é um algoritmo de ordenação não-comparativo que ordena números inteiros processando dígito por dígito.

LSD Radix Sort (Least Significant Digit)

  1. Começa pelo dígito menos significativo (unidades)
  2. Distribui os números em baldes de acordo com o valor deste dígito
  3. Coleta os números dos baldes na ordem (0-9)
  4. Repete o processo para cada posição de dígito (dezenas, centenas, etc.)
  5. O array final estará completamente ordenado

O LSD Radix Sort tem complexidade de tempo O(n·k), onde n é o número de elementos e k é o número de dígitos do maior número.
