Como Funciona o Radix Sort?

O Radix Sort é um algoritmo de ordenação não-comparativo que ordena números inteiros processando dígito por dígito. Existem duas variantes principais:

LSD Radix Sort (Least Significant Digit)

  1. Começa pelo dígito menos significativo (unidades)
  2. Distribui os números em baldes de acordo com o valor deste dígito
  3. Coleta os números dos baldes na ordem (0-9)
  4. Repete o processo para cada posição de dígito (dezenas, centenas, etc.)
  5. O array final estará completamente ordenado

MSD Radix Sort (Most Significant Digit)

  1. Começa pelo dígito mais significativo (dígito mais à esquerda)
  2. Distribui os números em baldes de acordo com o valor deste dígito
  3. Para cada balde não vazio, ordena recursivamente usando o próximo dígito mais significativo
  4. O resultado final é o conjunto de todos os baldes concatenados em ordem

Diferenças: O LSD processa todos os números de uma vez para cada dígito, do menos para o mais significativo, enquanto o MSD trabalha recursivamente, dividindo o problema em subproblemas menores, do dígito mais significativo para o menos. O LSD precisa processar todos os dígitos para garantir a ordenação, enquanto o MSD pode terminar antes em alguns grupos.

Ambos têm complexidade de tempo O(n·k), onde n é o número de elementos e k é o número de dígitos do maior número.
